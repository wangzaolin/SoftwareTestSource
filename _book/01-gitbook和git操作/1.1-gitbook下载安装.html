1.1-gitbook下载安装

gitbook下载安装

1）node.js下载安装

首先需要下载node.js并安装，因为gitbook需要依赖node.js的环境，这个可以直接在官网下载，默认安装即可；

2）gitbook命令行工具下载安装

在cmd窗口，输入命令：

npm install -g gitbook-cli

安装验证：

在cmd窗口输入命令：gitbook -V
